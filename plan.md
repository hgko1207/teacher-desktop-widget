### Step 3: 설정 모달 확장 🟡 높음

#### 3-7. 학교 정보 설정
- **구현**: 설정 모달에 "학교 설정" 탭 추가
- **기능**: 학교명, 학년/반, 지역 (날씨용), 학생 수
- **우선순위**: 🟡 높음

#### 3-8. 교시 시간 편집
- **구현**: 설정 모달에서 각 교시 시작/종료 시간 직접 편집
- **기능**: 1~7교시 각각의 시작/종료 시간 입력 필드
- **중요**: 이게 있어야 시간표/현재수업/진행률바가 정확해짐
- **우선순위**: 🟡 높음

#### 3-9. 퇴근 시간 설정
- **구현**: 설정 모달에서 퇴근 시간 변경 (기본 16:30)
- **우선순위**: 🟢 보통

#### 3-10. 즐겨찾기 런처 편집
- **구현**: 설정 모달에서 퀵 링크 추가/편집/삭제
- **기능**: 이름 + URL + 색상 선택
- **우선순위**: 🟢 보통

### Step 4: API 연동 🟢 보통 (Step 1~3 완료 후)

#### 3-11. 나이스 급식 API
- **API**: `https://open.neis.go.kr/hub/mealServiceDietInfo`
- **구현**: Electron 메인 프로세스에서 API 호출 → IPC로 렌더러에 전달
- **기능**: 학교 코드 설정 → 오늘 급식 자동 로드 → 캐시 (하루 1회)
- **우선순위**: 🟢 보통

#### 3-12. 기상청 날씨 API
- **API**: 기상청 단기예보 + 에어코리아
- **구현**: 지역 설정 → 현재 온도/날씨/미세먼지 → 1시간 캐시
- **우선순위**: 🟢 보통

#### 3-13. 나이스 시간표 API
- **API**: `/hub/misTimetable` 등
- **구현**: 학교 코드 + 학년/반 → 주간 시간표 자동 로드
- **우선순위**: 🔵 낮음
