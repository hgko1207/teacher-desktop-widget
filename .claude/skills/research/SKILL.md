name: research
description: 기술 리서치, 시장 조사, 레퍼런스 분석 결과를 research.md에 반영
user-invocable: true
argument-hint: [리서치 주제]
---

사용자의 리서치 요청을 처리합니다.

1. 요청된 주제를 웹 검색 및 분석합니다: $ARGUMENTS
2. 현재 research.md를 읽어 기존 리서치 내용을 파악합니다.
3. 새로운 리서치 결과를 적절한 섹션에 추가합니다:
   - 기술 관련 → 기술 스택 분석 섹션
   - 시장/경쟁 → 경쟁 제품 분석 섹션
   - 레퍼런스 → 레퍼런스 분석 섹션
   - 교사 니즈 → 업무 환경 분석 섹션
4. 출처를 반드시 포함합니다.
5. plan.md에 반영이 필요한 내용은 별도 안내합니다.
