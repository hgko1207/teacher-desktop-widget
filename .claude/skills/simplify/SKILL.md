user-invocable: true
argument-hint: [파일 경로 또는 영역]
---

변경된 코드를 검토하고 개선합니다.

1. 지정된 파일 또는 최근 변경 사항을 분석합니다: $ARGUMENTS
2. 다음 항목을 검토합니다:
   - 코드 중복 여부 → 공통 유틸/훅으로 추출
   - any/unknown 타입 사용 여부 → 구체적 타입으로 교체
   - 불필요한 복잡도 → 단순화
   - 미사용 코드 → 제거
   - 성능 이슈 → 메모이제이션, 지연 로딩 등
3. 발견된 문제를 수정합니다.
4. typecheck를 실행하여 새로운 에러가 없는지 확인합니다.
