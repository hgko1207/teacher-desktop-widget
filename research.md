# 교사용 윈도우 데스크탑 위젯 - 리서치 문서

> 작성일: 2026-03-19
> 목적: 교사(아내)를 위한 윈도우 데스크탑 위젯 개발을 위한 종합 리서치

---

## 1. 프로젝트 배경 및 목표

### 1.1 배경
- 한국 교사들의 행정업무 시간은 주당 **7.23시간** (2022년 기준), 2013년 대비 **27.9% 증가**
- 반면 수업시간은 18.72시간 → 16.47시간으로 **감소** (수업에 집중하기 어려운 구조)
- 나이스(NEIS) 4세대 전환 후에도 기존 불편사항 미개선 + 새로운 불편 가중
- 교육부는 2025~2026년 디지털 행정혁신, AI 활용 업무경감 정책 추진 중

### 1.2 목표
- 교사의 **일상 업무 효율**을 높이는 윈도우 데스크탑 위젯 개발
- 데스크탑에서 항상 빠르게 접근 가능한 **올웨이즈-온-탑(Always on Top)** 위젯
- 깔끔하고 미니멀한 UI로 업무 흐름을 방해하지 않는 디자인
- 교사 특화 기능 제공 (시간표, 출결, 할일, 메모 등)

## 2. 한국 교사의 업무 환경 분석

### 2.1 교사가 느끼는 주요 불편사항
| 불편사항 | 상세 내용 |
|---------|----------|
| **나이스(NEIS) 사용 불편** | 개인정보 조회 시 매번 사유 입력 강제, 느린 속도, 복잡한 UI |
| **출결관리 이중입력** | 과목별 분산된 출결정보를 담임이 일일이 확인해야 함 |
| **세특 입력 부담** | 세부능력 및 특기사항 입력 시점의 유연성 부족 |
| **행정업무 과다** | 공문 처리, 각종 보고서, 설문 응답 등이 수업 준비 시간 압박 |
| **일정 관리 분산** | 학교 행사, 수업 시간표, 개인 일정이 여러 곳에 흩어져 있음 |
| **학부모 소통** | 알림장, 가정통신문 등 별도 채널 관리 필요 |

### 2.2 교사의 일과 흐름
```
07:30~08:30  출근 및 수업 준비
08:30~08:50  조회/아침활동
09:00~12:20  오전 수업 (1~4교시)
12:20~13:10  점심시간
13:10~15:10  오후 수업 (5~6교시)
15:10~16:30  방과후/행정업무/회의
16:30~       퇴근 준비, 다음 날 수업 준비
```

### 2.3 교사가 자주 사용하는 도구들
- **나이스(NEIS)** - 출결, 성적, 학적 관리 (필수)
- **컴시간 알리미** - 시간표 조회
- **학교종이** - 학교 일정 및 알림
- **슈퍼스쿨** - 출결관리, 학부모 소통
- **Google Classroom** - 과제 관리, 피드백
- **미리캔버스** - 교육용 디자인 자료 제작
- **ChatGPT / Notion AI** - 수업 아이디어, 보고서 초안

## 3. 위젯 기능 카테고리 (교사 맞춤)

### 3.1 핵심 위젯 (Must-Have)

#### (1) 시간표 위젯
- 오늘의 수업 시간표 한눈에 보기
- 현재 교시 하이라이트 + 다음 수업까지 남은 시간
- 요일별 자동 전환
- 교시별 과목, 반, 교실 정보 표시
- 시간표 데이터 수동 입력 또는 CSV 가져오기

#### (2) 할 일 목록 (To-Do) 위젯
- 체크리스트 형태의 업무 관리
- 카테고리 분류: `수업준비`, `행정업무`, `학생관리`, `기타`
- 우선순위(높음/보통/낮음) 설정
- 마감일 설정 및 D-day 표시
- 완료 항목 자동 정리

#### (3) 스티키 노트(메모) 위젯
- 빠른 메모 작성 (수업 중 떠오른 아이디어, 학생 상담 메모 등)
- 색상 분류 (과목별/용도별)
- 고정(Pin) 기능
- 검색 기능

#### (4) 디데이(D-Day) / 카운트다운 위젯
- 중요 일정 카운트다운: 중간고사, 기말고사, 수능, 방학, 학교 행사 등
- 여러 D-Day 동시 표시
- 학사일정 기반 자동 설정 옵션

#### (5) 시계 및 타이머 위젯
- 현재 시간 + 현재 교시 표시
- 포모도로 타이머 (25분 집중 + 5분 휴식)
- 수업 타이머 (남은 수업 시간 표시)
- 쉬는 시간 알림

### 3.2 보조 위젯 (Nice-to-Have)

#### (6) 학급 출석 간편 체크 위젯
- 학급 학생 목록 표시
- 원클릭 출석/결석/조퇴/지각 체크
- 일별 출석 현황 요약
- (나이스 연동은 어렵지만, 빠른 메모용으로 활용)

#### (7) 학사 일정 / 캘린더 위젯
- 월간/주간 미니 캘린더
- 학교 행사, 시험, 방학 등 주요 일정 표시
- 오늘의 일정 알림
- Google Calendar / Outlook 연동 가능

#### (8) 날씨 위젯
- 현재 날씨 + 오늘의 예보
- 운동장 수업/야외 활동 가능 여부 판단에 유용
- 미세먼지 정보 포함

#### (9) 급식 메뉴 위젯
- 오늘의 급식 메뉴 표시
- 학교 급식 API 연동 (나이스 급식 Open API 활용 가능)
- 알레르기 정보 표시

#### (10) 빠른 링크 위젯
- 자주 사용하는 사이트 바로가기
- 나이스, 에듀넷, 학교 홈페이지, 구글 클래스룸 등
- 커스터마이징 가능한 바로가기 버튼

### 3.3 확장 위젯 (Future)

#### (11) 학생 랜덤 뽑기 위젯
- 발표자/당번 랜덤 선정
- 학급 학생 목록 기반
- 이미 뽑힌 학생 제외 옵션

#### (12) 소음 측정기 / 집중도 위젯
- 마이크 입력으로 교실 소음 레벨 시각화
- 학생들에게 보여주는 집중도 게이지

#### (13) AI 어시스턴트 위젯
- 수업 아이디어 생성
- 평가 문항 초안 작성
- 학생 상담 기록 요약

## 4. 기술 스택 분석 및 추천

### 4.1 프레임워크 비교

| 기술 | 장점 | 단점 | 추천도 |
|------|------|------|--------|
| **Electron** | 웹 기술(HTML/CSS/JS) 사용, 풍부한 생태계, 투명/항상위 창 지원 | 메모리 사용량 높음 (~100MB+) | ★★★★☆ |
| **Tauri** | 경량 (~10MB), Rust 기반, 보안 우수 | 비교적 새로운 생태계, 학습 곡선 | ★★★★★ |
| **Flutter Desktop** | 아름다운 UI, 크로스 플랫폼, 핫 리로드 | 데스크탑 지원 성숙도 부족, 위젯 특화 기능 부족 | ★★★☆☆ |
| **WPF (.NET)** | Windows 네이티브, 성능 우수, 위젯 구현 용이 | Windows 전용, C# 필요 | ★★★★☆ |
| **Rainmeter** | 기존 위젯 플랫폼, 커뮤니티 활발 | 커스텀 기능 구현 제한적, 스크립트 언어 비직관적 | ★★☆☆☆ |

### 4.2 추천 기술 스택

#### Option A: Electron + React (빠른 개발, 풍부한 UI)
```
프레임워크: Electron
프론트엔드: React + TypeScript
UI 라이브러리: Tailwind CSS + Framer Motion
상태관리: Zustand
로컬 저장: electron-store (JSON 기반)
빌드 도구: Vite + electron-builder
```

**장점:**
- 웹 개발 경험 활용 가능
- 풍부한 UI 컴포넌트와 애니메이션
- 빠른 프로토타이핑
- Always-on-top, 투명 창, 시스템 트레이 지원 우수

#### Option B: Tauri + React (경량, 고성능) ← 권장
```
프레임워크: Tauri v2
프론트엔드: React + TypeScript
UI 라이브러리: Tailwind CSS
상태관리: Zustand
로컬 저장: Tauri fs API + JSON
빌드 도구: Vite
```

**장점:**
- 극도로 가벼운 바이너리 (~5-10MB vs Electron ~100MB+)
- 낮은 메모리 사용량 (교사 PC 사양 고려)
- 보안 우수 (Rust 기반 백엔드)
- Always-on-top, 투명 창 지원
- 자동 업데이트 내장

### 4.3 핵심 기술 요구사항

```
1. Always-on-Top 윈도우
   - 다른 창 위에 항상 표시
   - 토글 가능 (필요시 일반 창으로 전환)

2. 투명/반투명 배경
   - 데스크탑과 자연스럽게 어울리는 글래스 효과
   - 불투명도 조절 가능

3. 시스템 트레이 연동
   - 트레이 아이콘으로 최소화
   - 트레이에서 각 위젯 토글

4. 드래그 앤 드롭 배치
   - 위젯을 자유롭게 데스크탑에 배치
   - 위치 기억 및 복원

5. 자동 시작
   - Windows 시작 시 자동 실행
   - 가벼운 부팅으로 PC 속도 영향 최소화

6. 로컬 데이터 저장
   - 클라우드 불필요, 로컬 JSON/SQLite 저장
   - 개인정보 보호 (학생 정보 등)
```

## 5. UI/UX 디자인 방향

### 5.1 디자인 원칙
1. **미니멀리즘**: 불필요한 장식 제거, 정보에 집중
2. **글래스모피즘(Glassmorphism)**: 반투명 배경 + 블러 효과로 세련된 느낌
3. **부드러운 색상**: 눈의 피로를 줄이는 파스텔 / 뉴트럴 톤
4. **다크/라이트 모드**: 환경에 맞게 전환 가능
5. **컴팩트**: 작은 크기로 화면을 차지하지 않음
6. **직관적 인터랙션**: 클릭, 드래그 등 간단한 조작

## 7. 교사들이 좋아하는 항목/카테고리 분석

### 7.1 선호도 높은 기능 (우선순위순)
1. **시간표 자동 표시** - 매일 확인해야 하는 필수 정보
2. **할 일 체크리스트** - 업무 누락 방지, 성취감
3. **D-Day 카운트다운** - 시험, 행사 일정 한눈에
4. **급식 메뉴** - 매일 궁금해하는 정보
5. **빠른 메모** - 수업 중 빠른 기록
6. **날씨/미세먼지** - 야외활동 판단
7. **수업 타이머** - 시간 관리
8. **학생 랜덤 뽑기** - 수업 활용도 높음

### 7.2 교사 커뮤니티에서 인기 있는 디지털 도구 특징
- **간단한 조작**: 복잡한 설정 없이 바로 사용 가능
- **예쁜 디자인**: 교사들은 시각적으로 깔끔한 도구 선호
- **가벼움**: 학교 PC 사양이 낮은 경우가 많음 (중요!)
- **오프라인 사용**: 학교 네트워크 환경이 불안정할 수 있음
- **개인정보 보호**: 학생 정보가 외부로 유출되지 않아야 함
- **무료**: 교사 개인이 구매하기 어려운 경우 많음

### 7.3 교사 유형별 니즈
| 교사 유형 | 주요 니즈 |
|----------|----------|
| **초등 담임** | 시간표, 출석체크, 알림장, 학생 뽑기, 급식 |
| **중등 담임** | 시간표, 출결관리, D-Day(시험), 상담 메모 |
| **중등 교과** | 시간표(여러 반), 수업 타이머, 과제 관리 |
| **보직교사** | 할 일 목록, 일정 관리, 공문 마감 D-Day |
| **신규교사** | 모든 기능 + 업무 가이드 |

## 8. 경쟁 제품 분석

### 8.1 기존 교사용 도구
| 제품 | 장점 | 단점 |
|------|------|------|
| **컴시간 알리미** | 시간표 특화, 학교 연동 | 시간표만 제공, 디자인 구식 |
| **학교종이** | 다양한 기능, 모바일 지원 | 모바일 앱 위주, 데스크탑 위젯 없음 |
| **슈퍼스쿨** | 출결+소통 통합 | 웹 기반, 데스크탑 위젯 아님 |
| **TeacherKit** | 글로벌 서비스, 다기능 | 한국 학제 미지원, 유료 |
| **Windows 11 위젯** | OS 내장, 안정적 | 교사 특화 기능 없음 |

### 8.2 차별화 포인트
- **한국 교사 특화**: 한국 학사 일정, 나이스 API, 급식 연동
- **데스크탑 위젯**: 항상 화면에 보이는 형태 (웹/앱과 차별)
- **올인원**: 시간표 + 할일 + 메모 + 급식 + 날씨를 하나의 앱에서
- **경량**: 학교 PC에서도 부담 없이 실행
- **오프라인 우선**: 네트워크 없이도 기본 기능 사용 가능
- **무료 / 오픈소스**: 교사 커뮤니티에서 자유롭게 활용

### 11.1 릴스 내용 요약
교사가 **Windows 데스크탑 전체를 커스텀 대시보드**로 꾸민 것을 소개하는 영상.
배경화면(월페이퍼) 위에 **웹 기반 위젯 대시보드**를 올린 형태로, 교사 업무에 필요한 모든 정보를 한 화면에 배치.

### 11.2 확인된 위젯 구성요소 (스크린샷 기반 상세 분석)

#### A. 왼쪽 패널 - 바탕화면 빠른 폴더 (파일 정리 시스템)
```
┌─────────────────────────┐
│ 📁 바탕화면 빠른 폴더      │
│ "이 공간 위에 실제 윈도우   │
│  바탕화면 아이콘을 올려두고  │
│  사용하세요"               │
├─────────────────────────┤
│ 📁 2026 행복하길           │
│   2025 감주현 / 2024 안전생활부│
│   USER / PEKR SCAN        │
│   타른거처 / ☆가산점☆      │
│   inline AI / 루체름       │
├─────────────────────────┤
│ 📁 진행중 업무             │
│   "실제 아이콘들을 이 칸에   │
│    모아두세요"             │
│   → 새 폴더              │
├─────────────────────────┤
│ 📁 나중에 볼 파일          │
│   → 온등부 서류           │
├─────────────────────────┤
│ 📁 기타                   │
│   → 행구리                │
└─────────────────────────┘
```
- 바탕화면을 **카테고리별 구역**으로 나눠 파일/폴더를 정리
- "진행중 업무", "나중에 볼 파일", "기타" 등 **업무 상태별 분류**
- 점선 테두리로 드래그 영역 표시

#### B. 상단 영역 - 현재 시각 + 타이머
```
┌──────────────────┬──────────────┐
│ 현재 시각          │    07:33     │
│ 15:19             │  [+5분] [↺] [⏸]│
│ 2026년 3월 9일 월요일│              │
└──────────────────┴──────────────┘
```
- **큰 시계** (15:19) + 날짜 + 요일 (월요일 뱃지)
- **타이머** (07:33) - +5분 버튼, 리셋, 일시정지 버튼 포함
- 수업 시간 관리에 활용

- **교사 실제 업무**: 평가계획서, 자유학기 계획서, 운동부 현황, 소년체전 신청, 봉사활동 계획

### 11.5 레퍼런스에서 배울 점 & 우리 위젯에 반영할 사항

#### 반드시 반영해야 할 것
1. **주간 시간표 그리드** (가장 핵심!) - 반별 색상 코딩, 공강 시각화
2. **"지금 이 시간" 위젯** - 현재 몇 교시인지 + 공강/수업 표시 + 시간 범위
3. **급식 메뉴 + 칼로리** - 나이스 API 연동 필수
4. **D-Day 카운트다운** - 여름방학, 시험 등
5. **할 일 체크리스트** - 교사 실제 업무 (평가계획서, 제출 서류 등)
6. **날씨 + 미세먼지** - 뱃지 형태 등급 표시
7. **학사 일정** - 주간/월간 토글
8. **타이머** - 수업 시간 관리용 (+5분 버튼)
9. **간단 메모** - 오늘의 우선순위, 한 줄 메모

#### 차별화할 수 있는 개선 포인트
1. **항상 위에 떠있는 독립 위젯** (이 레퍼런스는 배경화면/웹 기반 → 다른 창에 가려짐)
2. **자동 데이터 연동** (나이스 API로 시간표/급식/학사일정 자동 로드)
3. **실시간 교시 하이라이트** (현재 교시 자동 강조 + 남은 시간 표시)
4. **알림 기능** (수업 시작 5분 전, 할 일 마감 등)
5. **설정 UI 개선** (레퍼런스는 설정 탭이 별도 → 인라인 편집 제공)
6. **다양한 학교급 프리셋** (초등/중등/고등/유치원)
7. **바탕화면 폴더 정리 가이드** 연동 (배경 이미지 자동 생성)
8. **위젯 개별 배치** (한 화면 고정이 아닌, 각 위젯을 자유롭게 배치)
